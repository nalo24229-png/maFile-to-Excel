# maFile-to-Excel
[EN] Bulk maFile &amp; proxy converter to XLSX for Steam Bot Farms and massive MarketApp imports. [RU] Массовый конвертер maFile данных и прокси в XLSX для ферм Steam ботов и быстрого импорта в MarketApp.

# maFile-to-Excel Converter [v1.0]

An automated tool designed to combine Steam `.maFile` data, accounts text logs, and proxies into a perfectly formatted Excel spreadsheet specifically tailored for **MarketApp** import. It eliminates manual input and saves hours of routine work for Steam Bot Farms.

Автоматический инструмент для объединения данных Steam `.maFile`, текстовых логов с аккаунтами и прокси в идеально отформатированную Excel-таблицу, созданную специально для импорта в **MarketApp**. Полностью избавляет от ручного ввода и экономит часы работы для владельцев ферм ботов.

---

## 🛠️ Main Features / Основные возможности

* **Trade URL Auto-Generation:** Automatically extracts SteamID from `.maFile` and calculates personal trade links if proxies are not used. 
* **Авто-генерация Trade URL:** Автоматически вытягивает SteamID из мафайлов и рассчитывает ссылки на обмен, если не используются прокси.
* **Smart Data Escaping:** Prevents broken XML structures caused by special characters (`&`, `<`, `>`, `"`) inside JSON logs.
* **Экранирование спецсимволов:** Защищает структуру таблицы от разрушения из-за кавычек и амперсандов внутри JSON-текста.
* **Dual Export Mode:** Generates one master table for mass import and separate configuration folders for every individual bot.
* **Двойной режим экспорта:** Создает общую таблицу для массового залива и отдельные папки с личными конфигами под каждого бота.

## 🚀 How To Use / Инструкция по запуску

### [EN] English Version
1. Download the English release of the software.
2. Place your `.maFile` documents inside `Input Data Here/mafiles/`.
3. Add your login and password database into `Input Data Here/accounts.txt` (format `login:password`).
4. (Optional) Put your proxy lines into `Input Data Here/proxies.txt`. If left empty, the script will automatically use generated Trade URLs instead.
5. Run `START.bat` in the root folder.
6. Take your finished tables from `Your Ready Accounts/` and upload them to MarketApp!

### [RU] Русская версия
1. Скачайте русскую сборку софта.
2. Поместите ваши файлы `.maFile` внутрь папки `Вписывать данные сюда/mafiles/`.
3. Добавьте базу ваших данных в `Вписывать данные сюда/accounts.txt` (в формате `логин:пароль`).
4. (Опционально) Закиньте список ваших прокси в `Вписывать данные сюда/proxies.txt`. Если оставить пустым, скрипт автоматически подставит сгенерированные ссылки на обмен.
5. Запустите файл `СТАРТ.bat` в корневой папке.
6. Заберите готовые таблицы из папки `Ваши готовые аккаунты/` и загрузите их в MarketApp!
